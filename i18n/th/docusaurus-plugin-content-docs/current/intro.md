# ยินดีต้อนรับสู่ EpubCraft

**EpubCraft** คือโปรแกรมแก้ไขและจัดพิมพ์ EPUB ระดับมืออาชีพที่ทำงานบนเบราว์เซอร์ของคุณ สร้าง แก้ไข และส่งออกอีบุ๊กที่จัดรูปแบบสวยงามด้วยตัวแก้ไขสองโหมด พรีวิวแบบเรียลไทม์ และเครื่องมือ AI

## EpubCraft คืออะไร?

EpubCraft คือแอปพลิเคชันบนเว็บที่สร้างขึ้นสำหรับนักเขียน สำนักพิมพ์ และผู้สร้างเนื้อหาที่ต้องการเวิร์กโฟลว์ที่มีประสิทธิภาพสำหรับการผลิตอีบุ๊ก EPUB มันรวมตัวแก้ไขโค้ดเข้ากับแผงพรีวิวแบบสด ให้คุณควบคุม HTML และ CSS ของหนังสือได้อย่างเต็มที่พร้อมเห็นการเปลี่ยนแปลงทันที

### ความสามารถหลัก

- **การแก้ไขสองโหมด** — เขียนใน HTML พร้อมไฮไลท์ซินแท็กซ์ หรือสลับไปโหมด CSS สำหรับจัดรูปแบบ
- **พรีวิวแบบเรียลไทม์** — ดูหนังสือของคุณในกรอบจำลองอุปกรณ์พร้อมซิงค์การเลื่อนสองทิศทาง
- **จัดการบท** — เพิ่ม จัดเรียง แบ่ง ทำซ้ำ และจัดระเบียบบทด้วยการลากและวาง
- **ฟอนต์และรูปภาพ** — อัปโหลดและฝังฟอนต์ TTF, OTF, WOFF, WOFF2 และรูปภาพ
- **สร้างเนื้อหาด้วย AI** — ใช้ Gemini AI สร้างบทหรือปรับปรุงข้อความ
- **ส่งออกอัจฉริยะ** — ส่งออกเป็น EPUB 2.0 พร้อมปรับแต่งข้อความไทย/เอเชีย หรือบันทึกเป็น ZIP
- **เครื่องมือ Regex** — ค้นหาและแทนที่ด้วยรูปแบบ regex แบ่งบทตามรูปแบบ
- **เทมเพลต CSS** — ใช้เทมเพลตในตัวหรือสร้างรูปแบบของคุณเอง
- **ตัวแก้ไขข้อมูลเมตา** — ตั้งชื่อเรื่อง ผู้แต่ง สำนักพิมพ์ ภาษา คำอธิบาย และภาพปก
- **รองรับเชิงอรรถ** — แทรกเชิงอรรถที่แสดงเป็นตัวเลขอ้างอิงใน EPUB
