# การจัดการบท

บทคือส่วนประกอบหลักของอีบุ๊ก EpubCraft มีเครื่องมือที่ยืดหยุ่นสำหรับจัดระเบียบบท

## การเพิ่มบท

มีหลายวิธีในการเพิ่มบท:

- **เพิ่มบทท้ายสุด** — คลิกปุ่มด้านล่างรายการบทในแถบด้านข้าง
- **แทรกด้านล่าง** — วางเมาส์บนบทใดก็ได้แล้วคลิกไอคอน **+** เพื่อแทรกบทใหม่หลังจากนั้น
- **เพิ่มหน้า** — ใช้ปุ่มเพิ่มหน้าที่ด้านบนของรายการบท:
  - **บทว่าง** — บทว่างที่ตำแหน่ง 1
  - **หน้าปก** — หน้าปกที่สร้างอัตโนมัติจากภาพปก
  - **ข้อมูลหนังสือ** — หน้าข้อมูล/เรื่องย่อที่สร้างจากข้อมูลเมตา

บทใหม่จะถูกสร้างพร้อมเนื้อหาเริ่มต้น:

```html
<h1>Chapter 1</h1>
<p>Start writing your book here...</p>
```

## เปลี่ยนชื่อบท

คลิกช่องชื่อบทในแถบเครื่องมือ (แสดงในโหมด HTML) เพื่อเปลี่ยนชื่อบทปัจจุบัน ชื่อจะแสดงในแถบด้านข้างและใช้เป็นหัวข้อบทในสารบัญของ EPUB ที่ส่งออก

## จัดเรียงบทใหม่

ลากบทในแถบด้านข้างโดยใช้ **ที่จับ** (ไอคอนจุดแนวตั้งทางซ้าย) วางที่ตำแหน่งที่ต้องการ ลำดับบทกำหนดลำดับการอ่านใน EPUB ที่ส่งออก

## ลบบท

วางเมาส์บนบทในแถบด้านข้างแล้วคลิกไอคอน **ถังขยะ** กล่องยืนยันจะปรากฏก่อนลบ คุณไม่สามารถลบบทสุดท้ายที่เหลือ — ต้องมีอย่างน้อยหนึ่งบทเสมอ

## ทำซ้ำเนื้อหา

แม้ไม่มีปุ่มทำซ้ำโดยเฉพาะ คุณสามารถ:
1. เลือกเนื้อหาทั้งหมดในตัวแก้ไข (`Ctrl+A`)
2. คัดลอก (`Ctrl+C`)
3. เพิ่มบทใหม่
4. วาง (`Ctrl+V`)

## แบ่งบท

ใช้เครื่องมือ **แบ่งบท** เพื่อแบ่งบท (หรือทุกบท) ที่รูปแบบเฉพาะ ดูรายละเอียดที่ [การแบ่งบท](/docs/advanced/chapter-splitting)

## หน้าเพิ่มเติม

EpubCraft สามารถสร้างหน้าพิเศษอัตโนมัติ:

### หน้าปก
สร้างหน้า HTML แสดงภาพปกจัดกลางเต็มความกว้าง ต้องตั้งค่าภาพปกในข้อมูลเมตาก่อน

### หน้าข้อมูลหนังสือ
สร้างหน้าข้อมูลที่มี:
- ชื่อหนังสือ
- ชื่อผู้แต่ง
- สำนักพิมพ์
- คำอธิบาย/เรื่องย่อ

ทั้งสองหน้าจะถูกแทรกที่ต้นรายการบท
