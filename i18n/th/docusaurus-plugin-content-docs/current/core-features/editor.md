# ตัวแก้ไข

ตัวแก้ไขของ EpubCraft คือตัวแก้ไขโค้ดพร้อมไฮไลท์ซินแท็กซ์สำหรับเขียนเนื้อหา HTML ของบทและรูปแบบ CSS

## โหมดตัวแก้ไข

### โหมด HTML

โหมดเริ่มต้นสำหรับแก้ไขเนื้อหาบท เขียน HTML มาตรฐานที่จะรวมอยู่ใน EPUB:

```html
<h1>ชื่อบท</h1>
<p>ข้อความย่อหน้าของคุณอยู่ที่นี่</p>
<p>ย่อหน้าอื่นพร้อม <strong>ตัวหนา</strong> และ <i>ตัวเอียง</i></p>
```

### โหมด CSS

สลับไปโหมด CSS โดยคลิก **Global CSS** ในแถบด้านข้าง ตัวแก้ไข CSS ใช้สไตล์ทั่วไปกับทุกบท:

```css
body {
  font-family: 'Noto Sans Thai', sans-serif;
  line-height: 1.8;
  margin: 1cm 0.8cm;
}

h1 {
  text-align: center;
  font-size: 2em;
}
```

## ไฮไลท์ซินแท็กซ์

ตัวแก้ไขใช้ Prism.js สำหรับไฮไลท์ซินแท็กซ์แบบเรียลไทม์:
- **โหมด HTML** — แท็ก แอตทริบิวต์ และค่าถูกแยกสี
- **โหมด CSS** — ตัวเลือก คุณสมบัติ และค่าถูกไฮไลท์

ตัวแก้ไขใช้ฟอนต์ monospace: Menlo, Monaco, Consolas, Liberation Mono, Courier New

## แถบเครื่องมือจัดรูปแบบ

เหนือตัวแก้ไข แถบเครื่องมือให้ปุ่มจัดรูปแบบด่วน:

| ปุ่ม | การกระทำ | คีย์ลัด |
|------|----------|---------|
| **B** | ครอบข้อความด้วยแท็ก `<b>` | — |
| *I* | ครอบข้อความด้วยแท็ก `<i>` | `Ctrl+I` |
| U | ครอบข้อความด้วยแท็ก `<u>` | `Ctrl+U` |
| ~~S~~ | ครอบข้อความด้วยแท็ก `<s>` | — |
| H1 | ครอบข้อความด้วยแท็ก `<h1>` | `Ctrl+1` |
| H2 | ครอบข้อความด้วยแท็ก `<h2>` | `Ctrl+2` |
| H3 | ครอบข้อความด้วยแท็ก `<h3>` | `Ctrl+3` |
| P | ครอบข้อความด้วยแท็ก `<p>` | `Ctrl+4` |
| ขนาดฟอนต์ | ครอบด้วย `<span style="font-size:...">` | — |
| รายการจุด | สร้าง `<ul>` พร้อม `<li>` | — |
| รายการตัวเลข | สร้าง `<ol>` พร้อม `<li>` | — |
| จัดซ้าย | ครอบด้วย `<div style="text-align:left">` | `Ctrl+5` |
| จัดกลาง | ครอบด้วย `<div style="text-align:center">` | `Ctrl+6` |
| จัดขวา | ครอบด้วย `<div style="text-align:right">` | `Ctrl+7` |
| เต็มแนว | ครอบด้วย `<div style="text-align:justify">` | `Ctrl+8` |
| รูปภาพ | แทรกรูปภาพ (อัปโหลดหรือ URL) | — |
| เชิงอรรถ | เปิดหน้าต่างแทรกเชิงอรรถ | — |
| AI ปรับปรุง | ปรับปรุงข้อความที่เลือกด้วย AI | — |

## เลิกทำ / ทำซ้ำ

ตัวแก้ไขเก็บประวัติพร้อม snapshot ทุก 700ms ใช้ปุ่ม **เลิกทำ** และ **ทำซ้ำ** ในแถบเครื่องมือ

## ตัดบรรทัด

สลับเปิด/ปิดการตัดบรรทัดด้วยปุ่มตัดบรรทัด เมื่อปิด บรรทัดยาวจะเลื่อนในแนวนอน

## ขนาดฟอนต์

เลือกขนาดฟอนต์แสดงผลจากดรอปดาวน์: 12px, 14px (ค่าเริ่มต้น), 16px, 18px, 20px, 24px, 30px หรือ 36px นี่มีผลเฉพาะการแสดงผลตัวแก้ไข ไม่ใช่ผลลัพธ์ EPUB

## จัดรูปแบบอัตโนมัติ

คลิกปุ่ม **Format** ในแถบเครื่องมือหลักเพื่อจัดย่อหน้าและทำความสะอาด HTML อัตโนมัติ
