# พรีวิว

แผงพรีวิวแสดงเนื้อหาบทที่เรนเดอร์แบบสดพร้อมสไตล์ทั้งหมดที่ใช้ แสดงในกรอบจำลองอุปกรณ์

## วิธีการทำงาน

พรีวิวเรนเดอร์เนื้อหาบทใน iframe แบบ sandbox พร้อม:

- **CSS ทั่วไป** ของคุณเป็นสไตล์ชีต
- **ฟอนต์ที่กำหนดเอง** ฉีดเป็นกฎ `@font-face` พร้อม data URI แบบ base64
- **รูปภาพ** แสดงจากข้อมูล base64 ที่ฝังไว้
- **เชิงอรรถ** แปลงจากเครื่องหมาย `[*]` เป็นตัวเลขอ้างอิง

พรีวิวอัปเดตอัตโนมัติขณะที่คุณพิมพ์ในตัวแก้ไข

## จำลองอุปกรณ์

พรีวิวแสดงในกรอบอุปกรณ์:
- ความกว้างสูงสุด: **600px**
- ความสูงขั้นต่ำ: **800px**
- เงาตกเพื่อรูปลักษณ์อุปกรณ์จริง

ให้มุมมองโดยประมาณว่าเนื้อหาจะดูเป็นอย่างไรบนแท็บเล็ตหรือเครื่องอ่านอีบุ๊ก

## ซิงค์การเลื่อนสองทิศทาง

ตัวแก้ไขและแผงพรีวิวถูกซิงโครไนซ์:

- **ตัวแก้ไข → พรีวิว**: ขณะเลื่อนตัวแก้ไข พรีวิวจะตามเพื่อแสดงเนื้อหาที่ตรงกัน
- **พรีวิว → ตัวแก้ไข**: คลิกที่องค์ประกอบในพรีวิวเพื่อข้ามไปบรรทัดซอร์สในตัวแก้ไข

การซิงค์ใช้แอตทริบิวต์ `data-source-line` ที่ฉีดเข้าไปใน HTML เพื่อแมปองค์ประกอบกลับไปยังหมายเลขบรรทัด

## คลิกเพื่อไปซอร์ส

คลิกที่ข้อความหรือองค์ประกอบในแผงพรีวิวเพื่อ:

1. ย้ายเคอร์เซอร์ตัวแก้ไขไปที่บรรทัดซอร์สที่ตรงกัน
2. ไฮไลท์บรรทัดเป้าหมายด้วยแอนิเมชันแฟลชสีเหลือง (0.5s ซ้ำ 3 ครั้ง)
3. เลื่อนตัวแก้ไขให้อยู่กลางบรรทัดเป้าหมาย

:::note
การคลิกลิงก์เชิงอรรถในพรีวิวจะนำทางเชิงอรรถแทนที่จะข้ามไปซอร์ส เพื่อให้คุณทดสอบการนำทางเชิงอรรถได้
:::

## พรีวิวในโหมด CSS

เมื่อแก้ไข CSS (โหมด Global CSS) พรีวิวแสดงบทที่เลือกพร้อมการเปลี่ยนแปลง CSS ของคุณแบบเรียลไทม์ ใช้ดรอปดาวน์ **บทพรีวิว** ในแถบเครื่องมือเพื่อสลับบทที่แสดง

## โหมดมุมมอง

| โหมด | คำอธิบาย |
|------|----------|
| **โค้ด** | ตัวแก้ไขเท่านั้น — พรีวิวถูกซ่อน |
| **แบ่งหน้าจอ** | ตัวแก้ไขและพรีวิวเคียงกัน |
| **พรีวิว** | พรีวิวเท่านั้น — ตัวแก้ไขถูกซ่อน |

มุมมองแบ่งหน้าจอเป็นโหมดที่แนะนำสำหรับแก้ไขเนื้อหา บนมือถือ มุมมองแบ่งหน้าจอไม่พร้อมใช้งาน — สลับระหว่างโหมดโค้ดและพรีวิวแทน
