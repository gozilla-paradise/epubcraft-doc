# การจัดรูปแบบ

EpubCraft มีปุ่มแถบเครื่องมือและคีย์ลัดสำหรับจัดรูปแบบเนื้อหาบทโดยไม่ต้องเขียน HTML เอง

## การจัดรูปแบบข้อความ

เลือกข้อความในตัวแก้ไขแล้วคลิกปุ่มจัดรูปแบบหรือใช้คีย์ลัดเพื่อครอบด้วยแท็ก HTML:

### รูปแบบอินไลน์

| รูปแบบ | ผลลัพธ์ HTML | ตัวอย่าง |
|--------|-------------|---------|
| ตัวหนา | `<b>ข้อความ</b>` | **ข้อความ** |
| ตัวเอียง | `<i>ข้อความ</i>` | *ข้อความ* |
| ขีดเส้นใต้ | `<u>ข้อความ</u>` | ข้อความ |
| ขีดทับ | `<s>ข้อความ</s>` | ~~ข้อความ~~ |

### ขนาดฟอนต์

เลือกข้อความ จากนั้นเลือกขนาดจากดรอปดาวน์ ครอบข้อความด้วย `<span>` พร้อม `font-size`:

```html
<span style="font-size: 24px;">ข้อความใหญ่</span>
```

ขนาดที่มี: 12px, 14px, 16px, 18px, 20px, 24px, 30px, 36px

## หัวข้อและย่อหน้า

| รูปแบบ | คีย์ลัด | ผลลัพธ์ HTML |
|--------|---------|-------------|
| หัวข้อ 1 | `Ctrl+1` | `<h1>ข้อความ</h1>` |
| หัวข้อ 2 | `Ctrl+2` | `<h2>ข้อความ</h2>` |
| หัวข้อ 3 | `Ctrl+3` | `<h3>ข้อความ</h3>` |
| ย่อหน้า | `Ctrl+4` | `<p>ข้อความ</p>` |

## การจัดตำแหน่งข้อความ

| การจัดตำแหน่ง | คีย์ลัด | ผลลัพธ์ HTML |
|--------------|---------|-------------|
| ซ้าย | `Ctrl+5` | `<div style="text-align: left">ข้อความ</div>` |
| กลาง | `Ctrl+6` | `<div style="text-align: center">ข้อความ</div>` |
| ขวา | `Ctrl+7` | `<div style="text-align: right">ข้อความ</div>` |
| เต็มแนว | `Ctrl+8` | `<div style="text-align: justify">ข้อความ</div>` |

## รายการ

### รายการจุด

คลิกปุ่มรายการจุดเพื่อครอบข้อความด้วยรายการไม่เรียงลำดับ ถ้าเลือกหลายบรรทัด แต่ละบรรทัดจะกลายเป็นรายการ:

```html
<ul>
  <li>รายการที่หนึ่ง</li>
  <li>รายการที่สอง</li>
</ul>
```

### รายการตัวเลข

เหมือนกันแต่เป็นรายการเรียงลำดับ:

```html
<ol>
  <li>รายการที่หนึ่ง</li>
  <li>รายการที่สอง</li>
</ol>
```

## รูปภาพ

คลิกปุ่ม **รูปภาพ** เพื่อแทรกรูปภาพ อัปโหลดไฟล์ภาพจากคอมพิวเตอร์ รูปภาพจะถูกเก็บเป็น base64 และฝังในเนื้อหาบท:

```html
<img src="data:image/png;base64,..." alt="image" />
```

ดูรายละเอียดเพิ่มเติมที่ [รูปภาพ](/docs/media/images)

## เชิงอรรถ

คลิกปุ่ม **เชิงอรรถ** เพื่อเปิดหน้าต่างเชิงอรรถ ป้อนข้อความเชิงอรรถแล้วคลิก **Insert Footnote** (หรือกด `Ctrl+Enter`) แทรกเครื่องหมายเชิงอรรถที่ตำแหน่งเคอร์เซอร์:

```html
<span class="footnote" data-footnote="ข้อความเชิงอรรถ">[*]</span>
```

ขณะส่งออก เครื่องหมายเหล่านี้จะถูกแปลงเป็นตัวเลขอ้างอิงอัตโนมัติพร้อมส่วนเชิงอรรถท้ายบท ดูรายละเอียดที่ [เชิงอรรถในการส่งออก](/docs/export/footnotes-in-export)

## รีเซ็ตการจัดรูปแบบ

คลิกปุ่ม **Reset** ในแถบเครื่องมือเพื่อลบรูปแบบอินไลน์และคลาสทั้งหมดจากเนื้อหาบทปัจจุบัน รวมถึงลบฟอนต์ที่อัปโหลดทั้งหมดและรีเซ็ต CSS เป็นค่าเริ่มต้น กล่องยืนยันจะปรากฏก่อนรีเซ็ต
