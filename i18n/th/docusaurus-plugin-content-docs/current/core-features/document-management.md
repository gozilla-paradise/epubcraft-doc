# การจัดการเอกสาร

EpubCraft มีเครื่องมือสำหรับสร้างหนังสือใหม่ นำเข้าเนื้อหาที่มีอยู่ และจัดการไฟล์โปรเจกต์

## สร้างหนังสือใหม่

คลิกปุ่ม **หนังสือใหม่** (ไอคอน FilePlus) ในแถบด้านข้างเพื่อเริ่มโปรเจกต์ใหม่ การรีเซ็ตครอบคลุม:

- บททั้งหมด (แทนที่ด้วยบทเริ่มต้นบทเดียว)
- ข้อมูลเมตา (รีเซ็ตเป็น "Untitled Book" โดย "Unknown Author")
- CSS ทั่วไป (รีเซ็ตเป็นเทมเพลตเริ่มต้น)
- ฟอนต์และรูปภาพที่อัปโหลดทั้งหมด
- ภาพปก

กล่องยืนยันจะปรากฏก่อนล้างงานของคุณ

## นำเข้าไฟล์

คลิกปุ่ม **นำเข้า** (ไอคอน FileUp) ในแถบด้านข้าง EpubCraft รองรับรูปแบบเหล่านี้:

### EPUB (.epub)

นำเข้าไฟล์ EPUB มาตรฐาน โดยแยก:
- **ข้อมูลเมตา** — ชื่อเรื่อง ผู้แต่ง สำนักพิมพ์ ภาษา คำอธิบาย จากไฟล์ OPF
- **บท** — ไฟล์เนื้อหา XHTML ทั้งหมดตามลำดับ spine
- **CSS** — สไตล์ชีตรวมเข้าในตัวแก้ไข CSS ทั่วไป
- **ฟอนต์** — ไฟล์ฟอนต์ที่ฝังไว้ (TTF, OTF, WOFF, WOFF2)
- **รูปภาพ** — รูปภาพทั้งหมดที่อ้างอิง แปลงเป็น base64
- **ปก** — ตรวจจับจาก OPF metadata หรือ cover-image property

ชื่อบทอ่านจากแท็ก `<h1>`, `<h2>`, `<h3>`, `<title>` หรือย่อหน้าแรกในแต่ละไฟล์

### ZIP (.zip)

นำเข้าข้อมูลสำรองโปรเจกต์ EpubCraft โครงสร้าง ZIP:
- `metadata.txt` — ข้อมูลเมตาของหนังสือ (รูปแบบ key: value)
- ไฟล์บท (HTML หรือ TXT เรียงตามตัวเลข)
- `global.css` — สไตล์ชีตทั่วไป
- `cover.jpg` / `cover.png` — ภาพปก
- ไฟล์ฟอนต์

### ข้อความ (.txt)

แปลงไฟล์ข้อความธรรมดาเป็นหนังสือ:
- แต่ละย่อหน้ากลายเป็นแท็ก `<p>`
- บรรทัดว่างแบ่งย่อหน้า
- ชื่อไฟล์กลายเป็นชื่อบท

### Markdown (.md)

แปลง Markdown เป็น HTML:
- `**ตัวหนา**` และ `*ตัวหนา*` → `<strong>`
- `__ตัวเอียง__` และ `_ตัวเอียง_` → `<i>`
- หัวข้อและย่อหน้าถูกรักษาไว้
- ชื่อไฟล์กลายเป็นชื่อบท

## สถานะโปรเจกต์

EpubCraft ทำงานทั้งหมดในหน่วยความจำ — โปรเจกต์ของคุณไม่ถูกบันทึกลงดิสก์โดยอัตโนมัติ ใช้ **ส่งออกเป็น ZIP** เพื่อบันทึกงานสำหรับภายหลัง คุณสามารถนำเข้า ZIP กลับเพื่อแก้ไขต่อได้
