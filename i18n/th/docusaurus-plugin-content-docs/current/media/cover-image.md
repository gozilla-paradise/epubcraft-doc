# ภาพปก

ภาพปกคือภาพแทนหนังสือของคุณ ปรากฏในคลังเครื่องอ่าน ร้านค้าออนไลน์ และสามารถแสดงเป็นหน้าแรกของอีบุ๊ก

## การตั้งค่าภาพปก

1. เปิด **ข้อมูลเมตาและปก** (ไอคอนตั้งค่าในแถบด้านข้าง หรือ `Ctrl+M`)
2. คลิกพื้นที่อัปโหลดเส้นประใต้ "Cover Image"
3. เลือกไฟล์ภาพจากคอมพิวเตอร์
4. ตัวอย่างปกปรากฏในหน้าต่าง — คลิก **X** เพื่อลบ
5. คลิก **Save Changes**

ภาพปกยังปรากฏเป็นตัวอย่างขนาดเล็กที่ด้านล่างของแถบด้านข้าง

## ภาพปกใน EPUB

เมื่อส่งออกเป็น EPUB ภาพปกจะ:

- แปลงเป็นรูปแบบ JPEG (พื้นหลังสีขาวสำหรับภาพโปร่งใส)
- บันทึกเป็น `OEBPS/Images/cover.jpg`
- อ้างอิงใน OPF manifest ด้วย `properties="cover-image"`
- อ้างอิงผ่าน `<meta name="cover" content="cover-image" />` ใน OPF metadata

## หน้าปก

คุณสามารถสร้างหน้าปกอัตโนมัติที่แสดงภาพปก:

1. คลิกปุ่ม **รูปภาพ** ที่ด้านบนรายการบทในแถบด้านข้าง
2. บทใหม่จะถูกแทรกที่ตำแหน่ง 1 พร้อม HTML ภาพปกจัดกลาง

## หน้าข้อมูลหนังสือ

ปุ่ม **Info** สร้างหน้าพร้อมข้อมูลเมตาของหนังสือ:

- ชื่อเรื่อง (เป็นหัวข้อ)
- ชื่อผู้แต่ง
- สำนักพิมพ์
- คำอธิบาย/เรื่องย่อ

หน้านี้ถูกแทรกที่ต้นรายการบท หลังหน้าปก (ถ้ามี)

## คำแนะนำ

- **ขนาด**: 1600 x 2560 พิกเซล (หรืออัตราส่วน 1:1.6 ใกล้เคียง) สำหรับความเข้ากันได้สูงสุด
- **รูปแบบ**: JPEG หรือ PNG (PNG จะถูกแปลงเป็น JPEG ขณะส่งออก)
- **ขนาดไฟล์**: แนะนำไม่เกิน 2MB
- **เนื้อหา**: ใส่ชื่อหนังสือและชื่อผู้แต่งบนปกเพื่อความสะดวกในการอ่านในคลังเครื่องอ่าน
