# ตัวแก้ไข CSS

EpubCraft มีตัวแก้ไข CSS ทั่วไปที่ควบคุมการจัดรูปแบบสำหรับทุกบทในหนังสือ สไตล์ที่คุณเขียนจะถูกใช้กับทุกบทขณะพรีวิวและส่งออก

## เข้าถึงตัวแก้ไข CSS

คลิก **Global CSS** ในแถบด้านข้างเพื่อสลับตัวแก้ไขเป็นโหมด CSS พื้นที่ตัวแก้ไขจะเปลี่ยนเป็นแสดงสไตล์ชีตทั่วไปพร้อมไฮไลท์ซินแท็กซ์ CSS

## การเขียน CSS

เขียน CSS มาตรฐานเพื่อจัดรูปแบบเนื้อหาหนังสือ CSS จะถูกใช้ภายในเอกสาร XHTML ของแต่ละบท:

```css
body {
  font-family: 'Georgia', serif;
  line-height: 1.8;
  margin: 1cm;
  color: #333;
}

h1 {
  text-align: center;
  font-size: 2em;
  font-weight: 700;
  margin-bottom: 1em;
}

p {
  text-indent: 2em;
  text-align: justify;
}

img {
  max-width: 100%;
  height: auto;
}
```

## พรีวิวแบบสด

การเปลี่ยนแปลง CSS จะแสดงในแผงพรีวิวทันที ในโหมด CSS แถบเครื่องมือจะแสดงดรอปดาวน์ **บทพรีวิว** — เลือกบทใดก็ได้เพื่อดูว่าสไตล์ของคุณดูเป็นอย่างไรกับเนื้อหาต่างกัน

## ฟอนต์ที่กำหนดเองใน CSS

ถ้าคุณอัปโหลดฟอนต์ผ่านตัวจัดการฟอนต์ อ้างอิงด้วยชื่อ font-family ที่สร้างขึ้น:

```css
body {
  font-family: 'MyCustomFont', serif;
}
```

ชื่อ font-family มาจากชื่อไฟล์ของฟอนต์ (ลบช่องว่าง) คุณสามารถดูชื่อที่แน่นอนในแผงตัวจัดการฟอนต์ การประกาศ `@font-face` ของฟอนต์จะถูกสร้างอัตโนมัติขณะส่งออก — คุณต้องอ้างอิงเฉพาะชื่อ font-family ใน CSS เท่านั้น

## CSS เริ่มต้น

EpubCraft มาพร้อมสไตล์ชีตเริ่มต้น (เทมเพลต Noto Sans Thai) ที่มี:

- การตั้งค่าฟอนต์ Noto Sans Thai
- ระยะขอบแบบ A4 (1cm บน/ล่าง, 0.8cm ซ้าย/ขวา)
- หัวข้อจัดกลางขนาด 2em
- ย่อหน้าเยื้อง (10mm)
- สไตล์เชิงอรรถ (ข้อความเล็กลง เส้นแบ่ง)
- ตัวหนา (700) และขีดเส้นใต้แทนตัวเอียง

## เคล็ดลับ

- ใช้หน่วยสัมพัทธ์ (`em`, `rem`, `%`) สำหรับขนาดฟอนต์เพื่อให้เคารพการตั้งค่าเครื่องอ่าน
- ตั้ง `max-width: 100%` บนรูปภาพเพื่อป้องกันการล้น
- ใช้ `text-align: justify` เพื่อรูปลักษณ์หนังสือที่เป็นมืออาชีพ
- กำหนด `page-break-before: always` บน `h1` เพื่อเริ่มบทในหน้าใหม่ในเครื่องอ่านที่รองรับ
