# เทมเพลต CSS

EpubCraft มีเทมเพลต CSS ในตัว 3 แบบเพื่อจัดรูปแบบหนังสือของคุณอย่างรวดเร็ว สามารถใช้เทมเพลตจากแถบเครื่องมือเมื่ออยู่ในโหมด CSS

## การใช้เทมเพลต

1. คลิก **Global CSS** ในแถบด้านข้างเพื่อเข้าโหมด CSS
2. คลิกดรอปดาวน์ **CSS Templates** ในแถบเครื่องมือ
3. เลือกเทมเพลต

CSS ของเทมเพลตจะแทนที่ CSS ทั่วไปปัจจุบัน ถ้าเทมเพลตต้องการฟอนต์เฉพาะ จะถูกดาวน์โหลดและเพิ่มในคลังฟอนต์อัตโนมัติ

:::caution
การใช้เทมเพลตจะเขียนทับ CSS ที่มีอยู่ คัดลอก CSS ปัจจุบันก่อนถ้าต้องการเก็บไว้
:::

## เทมเพลตที่มี

### Noto Sans Thai (ค่าเริ่มต้น)

เทมเพลตเริ่มต้น ปรับแต่งสำหรับเนื้อหาไทยและอังกฤษ

**ฟอนต์:** Noto Sans Thai Regular (ดาวน์โหลดอัตโนมัติ)

**สไตล์หลัก:**
- ขนาดหน้าแบบ A4 ระยะขอบ 1cm/0.8cm
- หัวข้อจัดกลาง (h1–h3) ขนาด 2em น้ำหนัก 700
- ย่อหน้าเยื้อง 10mm จัดเต็มแนว
- ความสูงบรรทัด: 1.8
- สไตล์เชิงอรรถพร้อมเส้นแบ่งด้านบน
- ตัวเอียงแสดงเป็นขีดเส้นใต้ (ไม่ใช่ตัวเอียง)

### Serif

เลย์เอาต์หนังสือคลาสสิกด้วยฟอนต์ serif แบบดั้งเดิม

**ฟอนต์:** Times New Roman / Georgia (ฟอนต์ระบบ — ไม่ต้องดาวน์โหลด), Helvetica สำหรับหัวข้อ

**สไตล์หลัก:**
- ความกว้างเนื้อหาสูงสุด: 65ch
- เลย์เอาต์จัดกลาง
- จัดเต็มแนว
- ชุดฟอนต์ระบบ
- ดีไซน์สะอาดเรียบง่าย

### Sarabun

ปรับแต่งสำหรับเนื้อหาภาษาไทยด้วยฟอนต์สารบัญ

**ฟอนต์:** Sarabun (ดาวน์โหลดอัตโนมัติ)

**สไตล์หลัก:**
- จัดรูปแบบหน้า A4 พร้อมระยะขอบเฉพาะ
- หัวข้อจัดกลางขนาด 2rem
- ย่อหน้าเยื้อง 10mm
- ขีดเส้นใต้แทนตัวเอียง (แท็ก `<em>`)
- น้ำหนักตัวหนา: 700

## การปรับแต่งเทมเพลต

หลังใช้เทมเพลต คุณสามารถแก้ไข CSS ได้อย่างอิสระ การปรับแต่งทั่วไป:

- เปลี่ยน `font-family` เป็นฟอนต์ที่อัปโหลด
- ปรับค่า `margin` และ `padding`
- แก้ไข `font-size` และ `line-height`
- เพิ่มคลาสที่กำหนดเองสำหรับส่วนพิเศษ
- เปลี่ยนการจัดตำแหน่งข้อความหรือการเยื้อง

## การดาวน์โหลดฟอนต์

เทมเพลตที่ต้องการฟอนต์ที่กำหนดเอง (Noto Sans Thai, Sarabun) จะดาวน์โหลดและติดตั้งฟอนต์อัตโนมัติเมื่อใช้ ฟอนต์จะปรากฏในตัวจัดการฟอนต์และถูกฝังใน EPUB ขณะส่งออก
