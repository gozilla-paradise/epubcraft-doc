# การเชื่อมต่อ AI

EpubCraft เชื่อมต่อกับ Gemini AI ของ Google เพื่อสร้างเนื้อหาบทและปรับปรุงข้อความ ฟีเจอร์นี้ต้องใช้ Gemini API key

## การตั้งค่า

### รับ API Key

1. เข้าไปที่ [Google AI Studio](https://aistudio.google.com/app/apikey)
2. เข้าสู่ระบบด้วยบัญชี Google
3. คลิก **Create API Key**
4. คัดลอกคีย์ที่สร้าง (เริ่มต้นด้วย `AIza...`)

### ตั้งค่าใน EpubCraft

1. คลิกไอคอน **กุญแจ** ในแถบด้านข้าง
2. วาง API key ในช่อง
3. คลิก **Save Key**

คีย์จะถูกเก็บใน localStorage ของเบราว์เซอร์ภายใต้คีย์ `gemini_api_key` ไม่เคยถูกส่งไปยังเซิร์ฟเวอร์อื่นนอกจาก Gemini API ของ Google

## การสร้างเนื้อหาด้วย AI

สร้างเนื้อหาบทใหม่จากคำสั่งข้อความ:

1. คลิกปุ่ม **AI Assist** ในแถบเครื่องมือ
2. พิมพ์คำสั่งในช่องที่ปรากฏ (เช่น "เขียนฉากลึกลับในห้องสมุดมืด")
3. กด **Enter** หรือคลิกปุ่มส่ง

AI จะสตรีมเนื้อหาเข้าบทปัจจุบัน ใช้อักขระ 1000 ตัวสุดท้ายของเนื้อหาที่มีเป็นบริบทเพื่อรักษาความต่อเนื่อง

### รายละเอียดการสร้าง

- **โมเดล**: gemini-3-flash-preview
- **โหมด**: สตรีมมิ่ง (เนื้อหาปรากฏทีละส่วน)
- **รูปแบบผลลัพธ์**: HTML (ย่อหน้า หัวข้อ ฯลฯ)
- **บริบท**: อักขระ 1000 ตัวสุดท้ายของบทปัจจุบัน

## การปรับปรุงข้อความด้วย AI

ปรับปรุงหรือแก้ไขข้อความที่เลือกด้วยคำสั่งเฉพาะ:

1. **เลือก** ข้อความที่ต้องการปรับปรุงในตัวแก้ไข
2. คลิกปุ่ม **AI Improve** ในแถบเครื่องมือจัดรูปแบบ
3. ป้อนคำสั่ง (เช่น "ทำให้ดราม่ามากขึ้น" "แก้ไวยากรณ์" "แปลเป็นภาษาไทย")
4. กด **Enter** หรือคลิกส่ง

AI จะแทนที่ข้อความที่เลือกด้วยเวอร์ชันที่ปรับปรุงแล้ว รักษาแท็ก HTML ที่มีอยู่

### รายละเอียดการปรับปรุง

- **โมเดล**: gemini-3-flash-preview
- **โหมด**: ไม่สตรีม (แทนที่ทั้งหมด)
- **พฤติกรรม**: บรรณาธิการมืออาชีพ — ทำตามคำสั่งอย่างแม่นยำ
- **รับรู้ HTML**: รักษาโครงสร้าง HTML ที่มีอยู่

## กรณีการใช้งาน

### สร้างเนื้อหา
- "เขียนฉากโรแมนติกระหว่างตัวละครหลักสองคน"
- "สร้างคำอธิบายรายละเอียดของปราสาท"
- "เขียนบทสนทนาระหว่างนักสืบกับผู้ต้องสงสัย"
- "สร้างบทนำสำหรับหนังสือทำอาหาร"

### ปรับปรุงข้อความ
- "แก้ไวยากรณ์และสะกดคำ"
- "ทำให้ย่อหน้านี้กระชับขึ้น"
- "เพิ่มรายละเอียดประสาทสัมผัส"
- "แปลเป็นภาษาไทยทางการ"
- "เปลี่ยนโทนเสียงให้ลึกลับมากขึ้น"
- "เขียนใหม่ในบุรุษที่หนึ่ง"

## ข้อจำกัด

- ต้องเชื่อมต่ออินเทอร์เน็ต
- API key ต้องถูกต้องและมีโควต้าเพียงพอ
- ควรตรวจสอบเนื้อหาที่สร้างเพื่อความถูกต้อง
- AI ใช้โมเดลทั่วไป — ความรู้เฉพาะทางอาจจำกัด
- อัตราจำกัดขึ้นอยู่กับแผน Google AI Studio
