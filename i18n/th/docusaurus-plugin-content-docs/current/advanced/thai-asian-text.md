# รองรับข้อความไทยและเอเชีย

EpubCraft มีการรองรับพิเศษสำหรับภาษาไทยและภาษาเอเชียอื่นๆ ที่ไม่ใช้ช่องว่างระหว่างคำ สิ่งนี้สำคัญสำหรับการตัดบรรทัดที่ถูกต้องบนเครื่องอ่าน

## ปัญหาการตัดบรรทัด

ภาษาไทย ลาว เขมร และอักษรเอเชียอื่นบางตัวไม่ใช้ช่องว่างระหว่างคำ เครื่องอ่านมาตรฐานไม่สามารถระบุจุดตัดบรรทัดได้ ทำให้:

- ข้อความล้นขอบหน้า
- ตัดบรรทัดกลางคำอย่างเก้ๆ กังๆ
- อ่านยาก

## การปรับแต่งอักขระความกว้างศูนย์

EpubCraft แก้ปัญหานี้โดยแทรก **อักขระความกว้างศูนย์** (Unicode `U+200B`) ที่ขอบเขตคำ อักขระที่มองไม่เห็นนี้บอกเครื่องอ่านว่าสามารถตัดบรรทัดได้อย่างปลอดภัยโดยไม่มีผลต่อข้อความที่มองเห็น

### วิธีเปิดใช้

1. คลิก **Export** (หรือกด `Ctrl+S`)
2. เลือกรูปแบบ **Standard EPUB**
3. ติ๊ก **Optimize Line Breaking**
4. คลิก **Export Now**

การปรับแต่งจะถูกประมวลผลผ่านบริการตัดคำภายนอกก่อนสร้าง EPUB

### วิธีการทำงาน

1. เนื้อหาข้อความของแต่ละบทถูกส่งไปยังบริการตัดคำ
2. บริการระบุขอบเขตคำโดยใช้กฎเฉพาะภาษา
3. อักขระความกว้างศูนย์ถูกแทรกที่แต่ละขอบเขต
4. เนื้อหาที่แก้ไขถูกใช้สำหรับ EPUB

เนื้อหาต้นฉบับในตัวแก้ไข **ไม่ถูกแก้ไข** — การปรับแต่งใช้กับไฟล์ที่ส่งออกเท่านั้น

## เทมเพลต CSS สำหรับภาษาไทย

เทมเพลต CSS ในตัวของ EpubCraft สองตัวออกแบบมาสำหรับเนื้อหาภาษาไทย:

### Noto Sans Thai (ค่าเริ่มต้น)
- ใช้ฟอนต์ Noto Sans Thai (ดาวน์โหลดอัตโนมัติ)
- ขีดเส้นใต้แทนตัวเอียง (ธรรมเนียมไทย)
- ความสูงบรรทัดที่เหมาะสม (1.8) สำหรับตัวอักษรไทย

### Sarabun
- ใช้ฟอนต์สารบัญ (ดาวน์โหลดอัตโนมัติ)
- เลย์เอาต์ปรับแต่งสำหรับ A4
- ย่อหน้าเยื้อง 10mm
- ขีดเส้นใต้แทนตัวเอียง

## ฟอนต์แนะนำสำหรับภาษาไทย

สำหรับการแสดงผลข้อความไทยที่ดีที่สุด ใช้ฟอนต์ที่ออกแบบมาสำหรับอักษรไทย:

- **Noto Sans Thai** — ฟอนต์สากลของ Google มาพร้อมเทมเพลต
- **Sarabun** — ฟอนต์ไทยสมัยใหม่สะอาด มาพร้อมเทมเพลต
- **TH Sarabun New** — ฟอนต์ไทยยอดนิยมสำหรับงานพิมพ์
- **Prompt** — ฟอนต์ไทยสมัยใหม่หลายน้ำหนัก

อัปโหลดฟอนต์ `.ttf` หรือ `.otf` ภาษาไทยผ่านตัวจัดการฟอนต์และอ้างอิงใน CSS

## เคล็ดลับ

- เปิด **Optimize Line Breaking** เสมอเมื่อส่งออกเนื้อหาภาษาไทย
- ใช้ `line-height: 1.8` หรือสูงกว่าสำหรับความอ่านง่ายของข้อความไทย
- ข้อความไทยมักดูดีกว่าด้วย `text-indent` มากกว่าระยะห่างระหว่างย่อหน้า
- แท็ก `<em>` ตามธรรมเนียมแสดงเป็นขีดเส้นใต้ (ไม่ใช่ตัวเอียง) ในภาษาไทย — เทมเพลตเริ่มต้นจัดการนี้โดยอัตโนมัติ
