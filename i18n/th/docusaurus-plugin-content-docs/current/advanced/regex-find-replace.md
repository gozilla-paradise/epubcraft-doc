# ค้นหาและแทนที่ด้วย Regex

EpubCraft มีเครื่องมือค้นหาและแทนที่ด้วย regex ที่ทรงพลังสำหรับแก้ไขเนื้อหาแบบกลุ่มข้ามบท

## เปิดเครื่องมือ

- คลิกปุ่ม **RegEx Replace** ในแถบเครื่องมือ (ใช้ได้ในโหมด HTML)
- หรือกด `Ctrl+H`

## การใช้ค้นหาและแทนที่

### รูปแบบค้นหา

ป้อน regular expression ของ JavaScript ในช่อง **Find (RegEx)** รูปแบบถูกใช้กับ flag `g` (global) จึงแทนที่ทุกที่ที่ตรง

ตัวอย่าง:
- `<p>\s*</p>` — ค้นหาย่อหน้าว่าง
- `<b>(.*?)</b>` — ค้นหาข้อความตัวหนา
- `\bword\b` — ค้นหาคำ "word" ทั้งคำ

### รูปแบบแทนที่

ป้อนข้อความแทนที่ คุณอ้างอิงกลุ่มที่จับได้:

- `$1` — กลุ่มจับที่ 1
- `$2` — กลุ่มจับที่ 2
- `$&` — ข้อความที่ตรงทั้งหมด

### ตัวอย่าง

| ค้นหา | แทนที่ | ผล |
|-------|--------|-----|
| `<b>(.*?)</b>` | `<strong>$1</strong>` | แปลง `<b>` เป็น `<strong>` |
| `<p>\s*</p>` | *(ว่าง)* | ลบย่อหน้าว่าง |
| `"(.*?)"` | `\u201C$1\u201D` | แปลงเครื่องหมายคำพูดตรงเป็นโค้ง |
| `\n{3,}` | `\n\n` | รวมบรรทัดว่างหลายบรรทัด |
| `<div[^>]*>(.*?)</div>` | `$1` | แกะแท็ก div |

## ขอบเขต

เลือกขอบเขตของการแทนที่:

- **บทปัจจุบัน** — มีผลเฉพาะบทที่ใช้งานอยู่
- **ทุกบท** — ใช้กับทุกบทในหนังสือ

## รูปแบบที่บันทึกไว้

คลิกไอคอน **หนังสือ** ข้างช่องค้นหาเพื่อเปิด **ตัวจัดการ Regex** ที่นี่คุณสามารถ:

- **บันทึก** รูปแบบที่ใช้บ่อยพร้อมชื่อ
- **โหลด** รูปแบบที่บันทึกไว้ลงในช่องค้นหา/แทนที่
- **แก้ไข** รูปแบบที่มีอยู่
- **ลบ** รูปแบบที่ไม่ต้องการแล้ว

รูปแบบที่บันทึกจะถูกเก็บใน localStorage ของเบราว์เซอร์และคงอยู่ระหว่างเซสชัน

## เคล็ดลับ

- ทดสอบ regex กับ **บทปัจจุบัน** ก่อนใช้กับ **ทุกบท**
- ใช้ `.*?` (ไม่โลภ) แทน `.*` (โลภ) เพื่อจับคู่ข้อความสั้นที่สุด
- จำไว้ว่าช่องค้นหาคาดหวัง regex ไม่ใช่ข้อความธรรมดา — escape อักขระพิเศษเช่น `.`, `(`, `)`, `[`, `]` ด้วย backslash
- ช่องแทนที่รองรับ `$1`, `$2` ฯลฯ สำหรับการอ้างอิงกลุ่มจับ
