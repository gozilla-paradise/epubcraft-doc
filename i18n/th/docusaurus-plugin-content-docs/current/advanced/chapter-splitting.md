# การแบ่งบท

ตัวแบ่งบทแบ่งบท (หรือทุกบท) เป็นหลายบทตามรูปแบบ regex ใช้ได้สำหรับแบ่งเอกสารยาวหรือจัดโครงสร้างเนื้อหาที่นำเข้าใหม่

## เปิดเครื่องมือ

คลิกปุ่ม **แบ่งบท** ในแถบเครื่องมือ (ใช้ได้เฉพาะโหมด HTML)

## วิธีการทำงาน

1. ป้อน **regular expression** ที่ตรงกับจุดที่ควรแบ่งบท
2. เลือก **ขอบเขต**: บทปัจจุบัน หรือ ทุกบท
3. เลือกเปิด **แปลงแท็กที่ตรงเป็น `<h1>`** (ไม่บังคับ)
4. คลิก **Split**

เนื้อหาจะถูกแบ่งที่แต่ละจุดที่ regex ตรง แต่ละส่วนที่แบ่งสร้างบทใหม่

## รูปแบบ Regex

รูปแบบเริ่มต้นคือ:

```
<p>(ตอนที่ .*?)</p>
```

ตรงกับหัวข้อบทภาษาไทยเช่น `<p>ตอนที่ 1 บทนำ</p>`

### กลุ่มจับ

ถ้า regex มีกลุ่มจับ (วงเล็บ) ข้อความที่จับได้จะกลายเป็น **ชื่อบท**:

| รูปแบบ | ตรงกับ | ชื่อบท |
|--------|--------|--------|
| `<p>(ตอนที่ .*?)</p>` | `<p>ตอนที่ 1 บทนำ</p>` | "ตอนที่ 1 บทนำ" |
| `<h1>(.*?)</h1>` | `<h1>Chapter One</h1>` | "Chapter One" |
| `<p>(Chapter \d+)</p>` | `<p>Chapter 5</p>` | "Chapter 5" |

ถ้าไม่มีกลุ่มจับ บทจะถูกกำหนดหมายเลขอัตโนมัติ

### แปลงเป็น H1

เมื่อเปิด **แปลงแท็กที่ตรงเป็น `<h1>`** องค์ประกอบที่ตรงจะถูกแทนที่ด้วยแท็ก `<h1>` ในบทใหม่ ใช้ได้เมื่อรูปแบบตรงกับแท็ก `<p>` ที่ควรเป็นหัวข้อ

## ขอบเขต

- **บทปัจจุบัน** — แบ่งเฉพาะบทที่ใช้งานอยู่
- **ทุกบท** — แบ่งทุกบทในหนังสือ

## กรณีการใช้งานทั่วไป

### แบ่งไฟล์ข้อความที่นำเข้า

หลังนำเข้าไฟล์ `.txt` ยาวเป็นบทเดียว:

```
<h1>(Chapter \d+.*?)</h1>
```

### แบ่งเนื้อหาภาษาไทย

สำหรับนิยายไทยที่มีเครื่องหมาย ตอนที่:

```
<p>(ตอนที่\s+\d+.*?)</p>
```

### แบ่งที่ตัวแบ่งหน้า

ถ้าเนื้อหาใช้ `<hr>` เป็นตัวแบ่งส่วน:

```
<hr\s*/?>
```

## รูปแบบที่บันทึก

คลิกไอคอน **หนังสือ** เพื่อเปิดตัวจัดการ Regex และบันทึกรูปแบบแบ่งที่ใช้บ่อย ตัวจัดการเก็บรูปแบบใน localStorage
