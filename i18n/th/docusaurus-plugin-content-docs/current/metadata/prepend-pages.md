# หน้าเพิ่มเติม

EpubCraft สามารถสร้างหน้าพิเศษอัตโนมัติและแทรกที่ต้นหนังสือ ใช้สำหรับสร้างหน้าส่วนนำที่ดูเป็นมืออาชีพ

## หน้าเพิ่มเติมที่มี

ปุ่มเพิ่มหน้าอยู่ที่ด้านบนของรายการบทในแถบด้านข้าง:

### บทว่าง

แทรกบทว่างที่ตำแหน่ง 1 พร้อมเนื้อหาเริ่มต้น:

```html
<h1>Chapter 1</h1>
<p>Start writing your book here...</p>
```

ใช้สำหรับเพิ่มหน้าอุทิศ คำนำ หรือส่วนนำอื่นๆ ที่คุณจะเขียนเอง

### หน้าปก

สร้างหน้า HTML แสดงภาพปก **ต้อง** ตั้งค่าภาพปกในตัวแก้ไขข้อมูลเมตาก่อน

HTML ที่สร้าง:

```html
<div style="text-align: center;">
  <img src="data:image/..." alt="Cover"
       style="max-width: 100%; height: auto;" />
</div>
```

สร้างภาพปกเต็มความกว้างจัดกลางเป็นหน้าแรกของหนังสือ

### หน้าข้อมูลหนังสือ

สร้างหน้าข้อมูลจากข้อมูลเมตาของหนังสือ:

- **ชื่อเรื่อง** แสดงเป็นหัวข้อจัดกลาง
- ชื่อ **ผู้แต่ง**
- **สำนักพิมพ์** (ถ้ามี)
- **คำอธิบาย** / เรื่องย่อ

ใช้เป็นหน้าปกในหรือหน้าชื่อเรื่องในหนังสือที่จัดพิมพ์

## เคล็ดลับการใช้งาน

- เพิ่ม **หน้าปก** ก่อน จากนั้น **หน้าข้อมูลหนังสือ** — สร้างลำดับส่วนนำมาตรฐาน (ปก → หน้าชื่อเรื่อง → เนื้อหา)
- หน้าที่สร้างเป็นบทปกติ — คุณแก้ไขได้อย่างอิสระหลังสร้าง
- หน้าเพิ่มเติมใช้สไตล์ CSS เดียวกับบทอื่นๆ
- คุณลบหน้าเพิ่มเติมได้เหมือนบทอื่นถ้าไม่ต้องการ
