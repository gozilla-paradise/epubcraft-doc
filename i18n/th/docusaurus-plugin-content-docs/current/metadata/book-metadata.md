# ข้อมูลเมตาของหนังสือ

ข้อมูลเมตาอธิบายคุณสมบัติของหนังสือและฝังอยู่ในไฟล์ EPUB เครื่องอ่านและร้านค้าออนไลน์ใช้ข้อมูลนี้แสดงหนังสือ

## การแก้ไขข้อมูลเมตา

เปิดตัวแก้ไขข้อมูลเมตา:
- คลิกไอคอน **ตั้งค่า** (ฟันเฟือง) ในแถบด้านข้าง
- หรือกด `Ctrl+M`

## ฟิลด์ข้อมูลเมตา

| ฟิลด์ | คำอธิบาย | ค่าเริ่มต้น | จำเป็น |
|-------|----------|------------|--------|
| **ชื่อเรื่อง** | ชื่อแสดงผลของหนังสือ | "Untitled Book" | ใช่ |
| **ผู้แต่ง** | ผู้สร้างหนังสือ | "Unknown Author" | ใช่ |
| **สำนักพิมพ์** | หน่วยงานจัดพิมพ์ | "Self-Published" | ไม่ |
| **ภาษา** | รหัสภาษา (เช่น `en`, `th`, `ja`) | "en" | ไม่ |
| **คำอธิบาย** | เรื่องย่อหรือบทสรุป | "A new book created with EpubCraft." | ไม่ |
| **ภาพปก** | ภาพปก (ดู [ภาพปก](/docs/media/cover-image)) | ไม่มี | ไม่ |

**วันที่** จะถูกตั้งค่าอัตโนมัติเป็นวันที่ปัจจุบันเมื่อสร้างหนังสือ

## ข้อมูลเมตาแมปไปยัง EPUB อย่างไร

ข้อมูลเมตาถูกเขียนลงในไฟล์ OPF (Open Packaging Format) ของ EPUB:

```xml
<metadata xmlns:dc="http://purl.org/dc/elements/1.1/"
          xmlns:opf="http://www.idpf.org/2007/opf">
  <dc:title>ชื่อหนังสือ</dc:title>
  <dc:creator opf:role="aut">ชื่อผู้แต่ง</dc:creator>
  <dc:publisher>ชื่อสำนักพิมพ์</dc:publisher>
  <dc:language>th</dc:language>
  <dc:description>คำอธิบายหนังสือ</dc:description>
  <dc:date>2026-03-20</dc:date>
  <dc:identifier id="BookId">urn:uuid:generated-uuid</dc:identifier>
  <meta name="generator" content="EpubCraft" />
</metadata>
```

## ข้อมูลเมตาในการส่งออก ZIP

เมื่อส่งออกเป็น ZIP ข้อมูลเมตาจะถูกบันทึกในไฟล์ `metadata.txt`:

```
title: ชื่อหนังสือ
creator: ชื่อผู้แต่ง
publisher: ชื่อสำนักพิมพ์
language: th
description: คำอธิบายหนังสือ
```

ไฟล์นี้จะถูกอ่านเมื่อนำเข้า ZIP กลับเพื่อกู้คืนข้อมูลเมตา

## เคล็ดลับ

- **รหัสภาษา**: ใช้แท็กภาษามาตรฐาน IETF รหัสทั่วไป: `en` (อังกฤษ), `th` (ไทย), `ja` (ญี่ปุ่น), `zh` (จีน), `ko` (เกาหลี)
- **ชื่อเรื่อง**: เขียนสั้นกระชับ — ชื่อยาวอาจถูกตัดในหน้าจอเครื่องอ่าน
- **คำอธิบาย**: เขียน 1–3 ประโยคสรุปเนื้อหาหนังสือ
