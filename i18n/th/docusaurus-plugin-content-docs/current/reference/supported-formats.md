# รูปแบบที่รองรับ

รายการรูปแบบไฟล์ทั้งหมดที่ EpubCraft รองรับสำหรับนำเข้า ส่งออก ฟอนต์ และรูปภาพ

## รูปแบบนำเข้า

| รูปแบบ | นามสกุล | คำอธิบาย |
|--------|---------|----------|
| EPUB | `.epub` | รูปแบบอีบุ๊กมาตรฐาน แยกบท ข้อมูลเมตา CSS ฟอนต์ รูปภาพ และปก |
| ZIP | `.zip` | ข้อมูลสำรองโปรเจกต์ EpubCraft กู้คืนสถานะโปรเจกต์ทั้งหมด |
| ข้อความ | `.txt` | ไฟล์ข้อความธรรมดา แปลงเป็น HTML ด้วยแท็ก `<p>` |
| Markdown | `.md` | ไฟล์ Markdown แปลงพื้นฐานเป็น HTML (ตัวหนา ตัวเอียง หัวข้อ) |

## รูปแบบส่งออก

| รูปแบบ | นามสกุล | คำอธิบาย |
|--------|---------|----------|
| EPUB 2.0 | `.epub` | รูปแบบอีบุ๊กมาตรฐาน เข้ากันได้กับเครื่องอ่านหลักทั้งหมด |
| Project ZIP | `.zip` | รูปแบบสำรอง มีบท (HTML หรือ TXT) CSS ข้อมูลเมตา ฟอนต์ และปก |

## รูปแบบฟอนต์

| รูปแบบ | นามสกุล | ประเภท MIME |
|--------|---------|------------|
| TrueType | `.ttf` | font/ttf |
| OpenType | `.otf` | font/otf |
| WOFF | `.woff` | font/woff |
| WOFF2 | `.woff2` | font/woff2 |

ฟอนต์ทุกรูปแบบสามารถอัปโหลดผ่านตัวจัดการฟอนต์และฝังในการส่งออก EPUB

## รูปแบบรูปภาพ

| รูปแบบ | นามสกุล | ประเภท MIME |
|--------|---------|------------|
| JPEG | `.jpg`, `.jpeg` | image/jpeg |
| PNG | `.png` | image/png |
| GIF | `.gif` | image/gif |
| SVG | `.svg` | image/svg+xml |
| WebP | `.webp` | image/webp |

รูปภาพถูกเก็บเป็น data URI base64 ในเนื้อหาบทและแยกเป็นไฟล์แยกขณะส่งออก EPUB

## ภาพปก

ภาพปกถูกแปลงเป็น **JPEG** ขณะส่งออก EPUB ไม่ว่ารูปแบบต้นฉบับ พื้นที่โปร่งใสถูกเติมด้วยพื้นหลังสีขาว

## รูปแบบภายใน EPUB

EPUB 2.0 ที่ส่งออกใช้รูปแบบภายในเหล่านี้:

| ไฟล์ | รูปแบบ | คำอธิบาย |
|------|--------|----------|
| บท | XHTML 1.1 | เนื้อหาบทแปลงจาก HTML |
| สไตล์ชีต | CSS | CSS ทั่วไปพร้อมกฎ @font-face |
| แพ็คเกจ | OPF 2.0 | Manifest ข้อมูลเมตา และ spine |
| การนำทาง | NCX | สารบัญ |
| คอนเทนเนอร์ | XML | META-INF/container.xml |
