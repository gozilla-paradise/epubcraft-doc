# คีย์ลัด

รายการคีย์ลัดทั้งหมดที่ใช้ได้ใน EpubCraft บน macOS ใช้ `Cmd` แทน `Ctrl`

## คีย์ลัดทั่วไป

| คีย์ลัด | การกระทำ |
|---------|----------|
| `Ctrl+S` | เปิดหน้าต่างส่งออก |
| `Ctrl+M` | เปิดตัวแก้ไขข้อมูลเมตา |
| `Ctrl+H` | เปิด RegEx Replace |

## คีย์ลัดตัวแก้ไข

### จัดรูปแบบข้อความ

| คีย์ลัด | การกระทำ |
|---------|----------|
| `Ctrl+I` | ครอบข้อความด้วยแท็ก `<i>` |
| `Ctrl+U` | ครอบข้อความด้วยแท็ก `<u>` |

### หัวข้อและโครงสร้าง

| คีย์ลัด | การกระทำ |
|---------|----------|
| `Ctrl+1` | ครอบข้อความด้วยแท็ก `<h1>` |
| `Ctrl+2` | ครอบข้อความด้วยแท็ก `<h2>` |
| `Ctrl+3` | ครอบข้อความด้วยแท็ก `<h3>` |
| `Ctrl+4` | ครอบข้อความด้วยแท็ก `<p>` |

### จัดตำแหน่งข้อความ

| คีย์ลัด | การกระทำ |
|---------|----------|
| `Ctrl+5` | ครอบด้วย div จัดซ้าย |
| `Ctrl+6` | ครอบด้วย div จัดกลาง |
| `Ctrl+7` | ครอบด้วย div จัดขวา |
| `Ctrl+8` | ครอบด้วย div เต็มแนว |

## คีย์ลัดหน้าต่าง

| คีย์ลัด | บริบท | การกระทำ |
|---------|-------|----------|
| `Ctrl+Enter` | หน้าต่างเชิงอรรถ | แทรกเชิงอรรถและปิด |
| `Enter` | ช่อง AI prompt | ส่งคำสั่ง AI |
| `Escape` | ช่อง AI prompt | ยกเลิกคำสั่ง AI |

## คีย์ลัดเบราว์เซอร์มาตรฐาน

คีย์ลัดมาตรฐานเหล่านี้ทำงานในตัวแก้ไขเช่นกัน:

| คีย์ลัด | การกระทำ |
|---------|----------|
| `Ctrl+A` | เลือกเนื้อหาทั้งหมด |
| `Ctrl+C` | คัดลอกที่เลือก |
| `Ctrl+V` | วางจากคลิปบอร์ด |
| `Ctrl+X` | ตัดที่เลือก |
| `Ctrl+Z` | เลิกทำ (เบราว์เซอร์เนทีฟ) |
