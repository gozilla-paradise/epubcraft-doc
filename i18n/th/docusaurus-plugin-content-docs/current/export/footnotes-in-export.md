# เชิงอรรถในการส่งออก

EpubCraft รองรับเชิงอรรถที่แทรกเป็นเครื่องหมายอินไลน์ในตัวแก้ไขและแปลงเป็นตัวเลขอ้างอิงขณะส่งออก

## วิธีการทำงาน

### ในตัวแก้ไข

เชิงอรรถถูกเก็บเป็น span พร้อมแอตทริบิวต์ `data-footnote`:

```html
<p>นี่คือข้อความพร้อมเชิงอรรถ<span class="footnote" data-footnote="เนื้อหาเชิงอรรถ">[*]</span> ตรงกลาง</p>
```

เครื่องหมาย `[*]` มองเห็นได้ในตัวแก้ไขและพรีวิวเพื่อระบุว่ามีเชิงอรรถ

### ใน EPUB ที่ส่งออก

ขณะส่งออก เชิงอรรถถูกแปลงเป็นตัวเลขอ้างอิง:

**ตัวอ้างอิงในข้อความ:**
```html
<p>นี่คือข้อความพร้อมเชิงอรรถ<sup class="footnote-marker">
  <a id="fnref1" href="#fn1">[1]</a>
</sup> ตรงกลาง</p>
```

**ส่วนเชิงอรรถ (ท้ายบท):**
```html
<div class="footnotes" epub:type="footnotes">
  <hr />
  <ol>
    <li id="fn1">
      <p>เนื้อหาเชิงอรรถ
        <a href="#fnref1" class="footnote-backref">↩</a>
      </p>
    </li>
  </ol>
</div>
```

## การนับเชิงอรรถ

เชิงอรรถนับตามลำดับภายในแต่ละบท เริ่มที่ 1 แต่ละบทมีการนับเชิงอรรถแยกกัน — เชิงอรรถ 1 ในบท 3 เป็นอิสระจากเชิงอรรถในบทอื่น

## การนำทาง

EPUB ที่ส่งออกรองรับการนำทางเชิงอรรถสองทิศทาง:

- **ไปข้างหน้า**: คลิก `[1]` ในข้อความข้ามไปเชิงอรรถด้านล่าง
- **ย้อนกลับ**: คลิก `↩` ในเชิงอรรถข้ามกลับไปจุดอ้างอิง

ใช้ลิงก์ anchor HTML มาตรฐาน (`#fn1` / `#fnref1`)

## การแทรกเชิงอรรถ

1. วางเคอร์เซอร์ที่ตำแหน่งที่ต้องการ
2. คลิกปุ่ม **เชิงอรรถ** ในแถบเครื่องมือ
3. พิมพ์เนื้อหาเชิงอรรถ
4. คลิก **Insert Footnote** หรือกด `Ctrl+Enter`

## สไตล์ CSS

เทมเพลต CSS เริ่มต้นมีสไตล์เชิงอรรถ:

```css
.footnotes {
  margin-top: 2em;
  border-top: 1px solid #ccc;
  padding-top: 1em;
  font-size: 0.85em;
}

.footnote-marker {
  font-size: 0.75em;
  vertical-align: super;
}

.footnote-backref {
  text-decoration: none;
  margin-left: 0.5em;
}
```

คุณปรับแต่งสไตล์เหล่านี้ได้ในตัวแก้ไข CSS

## โครงสร้าง HTML5

ผลลัพธ์เชิงอรรถใช้แอตทริบิวต์ HTML5 เชิงความหมาย:
- `epub:type="footnotes"` บน div ตัวบรรจุสำหรับการเข้าถึง EPUB
- รายการ `<ol>` มาตรฐานสำหรับเชิงอรรถแบบนับ
- ID anchor ตามรูปแบบ `fn{N}` และ `fnref{N}`

## ความเข้ากันได้กับเครื่องอ่าน

ลิงก์เชิงอรรถทำงานในเครื่องอ่านหลักทั้งหมด:
- Kindle (เชิงอรรถแบบป๊อปอัปในบางอุปกรณ์)
- Apple Books
- Kobo
- Google Play Books
- Calibre viewer

เครื่องอ่านบางตัวแสดงเชิงอรรถเป็นป๊อปอัปแทนที่จะนำทางไปท้ายหน้า
