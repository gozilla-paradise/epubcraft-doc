# ส่งออก ZIP

รูปแบบ ZIP โปรเจกต์คือรูปแบบสำรองของ EpubCraft บันทึกโปรเจกต์ทั้งหมดในไฟล์บีบอัดที่นำเข้ากลับได้

## การส่งออก

1. กด `Ctrl+S` หรือคลิกปุ่ม **Export**
2. เลือก **Project ZIP**
3. เลือกรูปแบบบท:
   - **HTML** — รักษาเนื้อหา HTML เต็ม
   - **Plain Text (.txt)** — ลบ HTML ส่งออกข้อความธรรมดา
4. คลิก **Export Now**

## โครงสร้าง ZIP

```
project.zip
├── metadata.txt           (ข้อมูลเมตาหนังสือ)
├── global.css             (สไตล์ชีตทั่วไป)
├── cover.jpg              (ภาพปก ถ้ามี)
├── 001_Chapter_Title.html (หรือ .txt)
├── 002_Chapter_Title.html
├── ...
├── font1.ttf              (ฟอนต์ที่อัปโหลด)
└── font2.otf
```

### metadata.txt

มีข้อมูลเมตาหนังสือในรูปแบบ key-value:

```
title: ชื่อหนังสือ
creator: ชื่อผู้แต่ง
publisher: Self-Published
language: th
description: คำอธิบายสั้นๆ ของหนังสือ
```

### ไฟล์บท

บทมีหมายเลขตามลำดับ (`001_`, `002_`, ฯลฯ) พร้อมชื่อบทเป็นชื่อไฟล์ รูปแบบขึ้นกับการเลือก:

- **HTML**: เนื้อหา HTML เต็มตามที่เขียนในตัวแก้ไข
- **Plain Text**: ลบแท็ก HTML ย่อหน้าแยกด้วยบรรทัดว่าง รายการแปลงเป็นจุด (•)

### Global CSS

ไฟล์ `global.css` มีสไตล์ชีตทั่วไปตรงตามที่ปรากฏในตัวแก้ไข CSS

### ฟอนต์

ไฟล์ฟอนต์ที่อัปโหลดทั้งหมดรวมอยู่ที่ระดับรากของ ZIP

## การนำเข้ากลับ

เพื่อกู้คืนโปรเจกต์จาก ZIP:

1. คลิกปุ่ม **นำเข้า** ในแถบด้านข้าง
2. เลือกไฟล์ `.zip`
3. EpubCraft กู้คืน:
   - ข้อมูลเมตาทั้งหมดจาก `metadata.txt`
   - บทตามลำดับ (เรียงตามหมายเลขชื่อไฟล์)
   - CSS ทั่วไปจาก `global.css`
   - ภาพปก
   - ไฟล์ฟอนต์

## กรณีการใช้งาน

- **สำรอง**: บันทึกงานก่อนทำการเปลี่ยนแปลงใหญ่
- **ย้าย**: ย้ายโปรเจกต์ระหว่างอุปกรณ์หรือเบราว์เซอร์
- **ทำงานร่วมกัน**: แชร์โปรเจกต์กับคนอื่น
- **ควบคุมเวอร์ชัน**: เก็บ snapshot ของโปรเจกต์ในขั้นตอนต่างๆ
