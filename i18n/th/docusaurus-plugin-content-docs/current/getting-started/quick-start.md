# เริ่มต้นอย่างรวดเร็ว

สร้างอีบุ๊กเล่มแรกใน 5 นาที คู่มือนี้ครอบคลุมขั้นตอนสำคัญ: ตั้งค่าข้อมูลเมตา เขียนเนื้อหา และส่งออก

## ขั้นตอนที่ 1: ตั้งค่าข้อมูลเมตาของหนังสือ

1. คลิกไอคอน **ตั้งค่า** (ฟันเฟือง) ในแถบด้านข้าง หรือกด `Ctrl+M`
2. กรอก:
   - **ชื่อเรื่อง** — ชื่อหนังสือ
   - **ผู้แต่ง** — ชื่อผู้เขียน
   - **สำนักพิมพ์** — หน่วยงานจัดพิมพ์ (ไม่บังคับ)
   - **ภาษา** — เช่น `en`, `th`
   - **คำอธิบาย** — เรื่องย่อสั้นๆ
3. อัปโหลด **ภาพปก** โดยคลิกพื้นที่อัปโหลด (ไม่บังคับ)
4. คลิก **Save Changes**

## ขั้นตอนที่ 2: เขียนบทแรก

ตัวแก้ไขเปิดพร้อมบทเริ่มต้น เริ่มแก้ไขเนื้อหา HTML:

```html
<h1>บทที่ 1</h1>
<p>นี่คือจุดเริ่มต้นของเรื่องราว...</p>
<p>EpubCraft ทำให้การเขียนและจัดรูปแบบอีบุ๊กเป็นเรื่องง่าย</p>
```

ใช้ **แถบเครื่องมือจัดรูปแบบ** เหนือตัวแก้ไข:
- **B** / **I** / **U** — ตัวหนา ตัวเอียง ขีดเส้นใต้
- **H1**, **H2**, **H3** — หัวข้อ
- ปุ่มจัดตำแหน่ง — ซ้าย กลาง ขวา เต็มแนว
- ปุ่มรายการ — รายการจุดและรายการตัวเลข

## ขั้นตอนที่ 3: เพิ่มบทเพิ่มเติม

ในแถบด้านข้าง:
- คลิก **เพิ่มบทท้ายสุด** เพื่อเพิ่มบทใหม่
- วางเมาส์บนบทที่มีอยู่แล้วคลิกไอคอน **+** เพื่อแทรกบทด้านล่าง
- **ลาก** ที่จับเพื่อจัดเรียงบทใหม่

## ขั้นตอนที่ 4: จัดรูปแบบหนังสือ

สลับไปโหมด CSS โดยคลิก **Global CSS** ในแถบด้านข้าง EpubCraft มีเทมเพลตในตัว 3 แบบ:

1. **Noto Sans Thai** — ค่าเริ่มต้น เหมาะสำหรับข้อความไทยและอังกฤษ
2. **Serif** — รูปลักษณ์หนังสือคลาสสิกด้วย Times New Roman
3. **Sarabun** — ปรับแต่งสำหรับภาษาไทยด้วยรูปแบบ A4

เลือกเทมเพลตจากดรอปดาวน์ **CSS Templates** ในแถบเครื่องมือ หรือเขียนรูปแบบของคุณเอง

## ขั้นตอนที่ 5: พรีวิว

คลิกปุ่ม **Preview** ในแถบเครื่องมือ (หรือใช้มุมมองแบ่งหน้าจอ) เพื่อดูหนังสือในกรอบจำลองอุปกรณ์ พรีวิวอัปเดตแบบเรียลไทม์ขณะที่คุณแก้ไข

คลิกที่องค์ประกอบใดก็ได้ในพรีวิวเพื่อข้ามไปยังบรรทัดซอร์สที่ตรงกันในตัวแก้ไข

## ขั้นตอนที่ 6: ส่งออก

1. กด `Ctrl+S` หรือคลิกปุ่ม **Export**
2. เลือกรูปแบบ:
   - **Standard EPUB** — พร้อมสำหรับเครื่องอ่านอีบุ๊กและจัดจำหน่าย
   - **Project ZIP** — รูปแบบสำรองที่นำเข้ากลับได้
3. สำหรับข้อความไทย/เอเชีย ติ๊ก **Optimize Line Breaking** เพื่อเพิ่มอักขระความกว้างศูนย์
4. คลิก **Export Now**

ไฟล์จะดาวน์โหลดอัตโนมัติ เพียงเท่านี้ — คุณสร้างอีบุ๊กด้วย EpubCraft แล้ว!
