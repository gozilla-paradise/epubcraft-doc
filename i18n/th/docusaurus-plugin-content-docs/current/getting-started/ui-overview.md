# ภาพรวม UI

อินเทอร์เฟซของ EpubCraft แบ่งเป็นสี่ส่วนหลัก: แถบด้านข้าง แถบเครื่องมือ ตัวแก้ไข และแผงพรีวิว

│          │          แถบเครื่องมือ                    │

│ ข้าง     │    ตัวแก้ไข       │      พรีวิว          │
│          │                  │                     │
│          │                  │                     │

## แถบด้านข้าง (แผงซ้าย)

แถบด้านข้างให้การนำทางและจัดการระดับหนังสือ:

- **ชื่อแอป** — แสดง "EpubCraft" พร้อมหมายเลขเวอร์ชัน
- **หนังสือใหม่** — เริ่มโปรเจกต์ใหม่ (ล้างทุกอย่าง)
- **นำเข้า** — โหลดไฟล์ EPUB, ZIP, TXT หรือ Markdown
- **รายการบท** — บททั้งหมดตามลำดับ คลิกเพื่อเลือก
  - ลากที่จับเพื่อจัดเรียงใหม่
  - วางเมาส์เพื่อแสดงปุ่มแทรก (+) และลบ (ถังขยะ)
- **เพิ่มบทท้ายสุด** — เพิ่มบทใหม่ที่ท้ายสุด
- **ตัวเลือกเพิ่มหน้า** — เพิ่มบทว่าง หน้าปก หรือหน้าข้อมูลหนังสือที่ด้านบน
- **Global CSS** — สลับตัวแก้ไขเป็นโหมด CSS
- **จัดการฟอนต์** — อัปโหลดและจัดการฟอนต์ที่กำหนดเอง
- **Gemini API Key** — ตั้งค่าการเชื่อมต่อ AI
- **ข้อมูลเมตาและปก** — เปิดตัวแก้ไขข้อมูลเมตา
- **ตัวอย่างปก** — แสดงภาพปกปัจจุบัน (ถ้ามี)

บนมือถือ แถบด้านข้างจะเลื่อนเข้ามาเป็นลิ้นชักและปิดเมื่อคุณเลือกบท

## แถบเครื่องมือ (แถบบน)

แถบเครื่องมือเปลี่ยนตามโหมดการแก้ไขปัจจุบัน:

### โหมด HTML
- ช่อง **ชื่อบท**
- ปุ่ม **โหมดมุมมอง**: โค้ด, แบ่งหน้าจอ, พรีวิว
- **รีเซ็ต** — ล้างการจัดรูปแบบ ฟอนต์ และ CSS
- **จัดรูปแบบ** — จัดย่อหน้า HTML อัตโนมัติ
- **RegEx Replace** — เปิดค้นหา/แทนที่ด้วย regex
- **แบ่งบท** — แบ่งบทตามรูปแบบ
- **AI Assist** — สร้างเนื้อหาด้วย Gemini AI
- **ส่งออก** — เปิดหน้าต่างส่งออก

### โหมด CSS
- ดรอปดาวน์ **CSS Templates** — ใช้เทมเพลตรูปแบบในตัว
- ตัวเลือก **บทพรีวิว** — เลือกบทที่จะแสดงพรีวิว
- ปุ่มโหมดมุมมองและส่งออก

## ตัวแก้ไข (แผงกลาง)

ตัวแก้ไขโค้ดพร้อมไฮไลท์ซินแท็กซ์:

- ไฮไลท์ซินแท็กซ์ HTML หรือ CSS (Prism.js)
- แถบเครื่องมือจัดรูปแบบ (ตัวหนา ตัวเอียง หัวข้อ รายการ จัดตำแหน่ง รูปภาพ เชิงอรรถ)
- ปุ่มเลิกทำ/ทำซ้ำ
- สลับการตัดบรรทัด
- ตัวเลือกขนาดฟอนต์ (12px–36px)
- ปุ่มปรับปรุงข้อความด้วย AI
- หมายเลขบรรทัด

## แผงพรีวิว (แผงขวา)

พรีวิวแบบเรียลไทม์ของบทปัจจุบัน:

- กรอบ **จำลองอุปกรณ์** (ความกว้างสูงสุด 600px)
- แสดงบทพร้อมสไตล์ CSS ทั้งหมด
- ฟอนต์ที่กำหนดเองแสดงผลผ่านกฎ `@font-face` ที่ฉีดเข้าไป
- รูปภาพแสดงจากข้อมูลที่ฝังไว้
- เชิงอรรถแปลงเป็นตัวเลขอ้างอิง
- คลิกที่องค์ประกอบเพื่อข้ามไปบรรทัดซอร์สในตัวแก้ไข
- ซิงค์การเลื่อนสองทิศทางกับตัวแก้ไข

## โหมดมุมมอง

| โหมด | คำอธิบาย |
|------|----------|
| **โค้ด** | ตัวแก้ไขอย่างเดียว เต็มความกว้าง |
| **แบ่งหน้าจอ** | ตัวแก้ไขและพรีวิวเคียงกัน |
| **พรีวิว** | พรีวิวอย่างเดียว เต็มความกว้าง |

มุมมองแบ่งหน้าจอไม่แสดงบนมือถือ — ใช้โหมดโค้ดหรือพรีวิวแทน
