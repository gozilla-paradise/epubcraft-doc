# การติดตั้ง

EpubCraft เป็นแอปพลิเคชันบนเว็บ — ไม่ต้องติดตั้ง เปิดในเบราว์เซอร์แล้วเริ่มสร้างอีบุ๊กได้ทันที

## เข้าใช้งาน EpubCraft

เข้าชมที่ URL เหล่านี้:

- **หลัก:** [epubcraft.vercel.app](https://epubcraft.vercel.app)
- **มิเรอร์:** [epubcraft.pages.dev](https://epubcraft.pages.dev)

## ความต้องการของระบบ

### รองรับเบราว์เซอร์

EpubCraft ทำงานได้ในเบราว์เซอร์สมัยใหม่ทุกตัวที่เปิดใช้งาน JavaScript:

- Google Chrome (แนะนำ)
- Mozilla Firefox
- Microsoft Edge
- Safari

### แอปเดสก์ท็อป (Tauri)

EpubCraft ยังมีให้ในรูปแบบแอปเดสก์ท็อปที่สร้างด้วย Tauri เวอร์ชันเดสก์ท็อปมีฟีเจอร์เหมือนกับเว็บแอปพร้อมการเข้าถึงระบบไฟล์แบบเนทีฟ

**แพลตฟอร์มที่รองรับ:**
- Windows 10/11
- macOS
- Linux

## การเปิดใช้งานครั้งแรก

เมื่อคุณเปิด EpubCraft ครั้งแรก แอปจะทำสิ่งต่อไปนี้โดยอัตโนมัติ:

1. สร้างหนังสือใหม่พร้อมข้อมูลเมตาเริ่มต้น ("Untitled Book" โดย "Unknown Author")
2. เพิ่มบทเริ่มต้นพร้อมเนื้อหาตัวอย่าง
3. ใช้เทมเพลต CSS Noto Sans Thai เริ่มต้น
4. เปิดในมุมมองแบ่งหน้าจอ (ตัวแก้ไข + พรีวิว เคียงกัน)

ตัวบ่งชี้สีน้ำเงินกะพริบบนปุ่ม **ข้อมูลเมตาและปก** ในแถบด้านข้างเตือนให้คุณตั้งค่าข้อมูลเมตาของหนังสือ ตัวบ่งชี้นี้จะหายไปหลังจากคุณโต้ตอบครั้งแรก

## ตัวเลือก: ตั้งค่า Gemini AI

เพื่อใช้ฟีเจอร์ AI (การสร้างเนื้อหาและปรับปรุงข้อความ) คุณต้องมี Google Gemini API key:

1. เข้าไปที่ [Google AI Studio](https://aistudio.google.com/app/apikey)
2. สร้าง API key (เริ่มต้นด้วย `AIza...`)
3. ใน EpubCraft คลิกไอคอน **กุญแจ** ในแถบด้านข้าง
4. วาง API key แล้วคลิก **Save Key**

คีย์จะถูกเก็บไว้ในเครื่องใน localStorage ของเบราว์เซอร์และไม่เคยถูกส่งไปยังเซิร์ฟเวอร์อื่นนอกจาก Gemini API ของ Google
