# Welcome to EpubCraft

**EpubCraft** is a professional EPUB editor and publisher that runs entirely in your browser. Create, edit, and export beautifully formatted ebooks with a powerful dual-mode editor, real-time preview, and AI-powered content tools.

## What is EpubCraft?

EpubCraft is a web-based application built for authors, publishers, and content creators who need a streamlined workflow for producing EPUB ebooks. It combines a code editor with a live preview panel, giving you full control over your book's HTML and CSS while seeing changes instantly.

### Key Capabilities

- **Dual-mode editing** — Write in HTML with syntax highlighting, or switch to CSS mode for styling
- **Live device preview** — See your book rendered in a device mockup with bidirectional scroll sync
- **Chapter management** — Add, reorder, split, duplicate, and organize chapters with drag-and-drop
- **Custom fonts & images** — Upload and embed TTF, OTF, WOFF, WOFF2 fonts and images directly
- **AI content generation** — Use Gemini AI to generate chapters or improve existing text
- **Smart export** — Export to EPUB 2.0 with Thai/Asian text optimization, or save as project ZIP
- **Regex tools** — Find and replace with regex patterns, split chapters by pattern
- **CSS templates** — Apply built-in templates or create your own styles
- **Metadata editor** — Set title, author, publisher, language, description, and cover image
- **Footnote support** — Insert footnotes that render as numbered references in the EPUB
