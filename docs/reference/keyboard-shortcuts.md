# Keyboard Shortcuts

A complete reference of keyboard shortcuts available in EpubCraft. On macOS, use `Cmd` instead of `Ctrl`.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+S` | Open Export dialog |
| `Ctrl+M` | Open Metadata editor |
| `Ctrl+H` | Open RegEx Replace |

## Editor Shortcuts

### Text Formatting

| Shortcut | Action |
|----------|--------|
| `Ctrl+I` | Wrap selection in `<i>` tags |
| `Ctrl+U` | Wrap selection in `<u>` tags |

### Headings & Structure

| Shortcut | Action |
|----------|--------|
| `Ctrl+1` | Wrap selection in `<h1>` tags |
| `Ctrl+2` | Wrap selection in `<h2>` tags |
| `Ctrl+3` | Wrap selection in `<h3>` tags |
| `Ctrl+4` | Wrap selection in `<p>` tags |

### Text Alignment

| Shortcut | Action |
|----------|--------|
| `Ctrl+5` | Wrap in left-aligned div |
| `Ctrl+6` | Wrap in center-aligned div |
| `Ctrl+7` | Wrap in right-aligned div |
| `Ctrl+8` | Wrap in justify-aligned div |

## Dialog Shortcuts

| Shortcut | Context | Action |
|----------|---------|--------|
| `Ctrl+Enter` | Footnote dialog | Insert footnote and close |
| `Enter` | AI prompt input | Submit AI prompt |
| `Escape` | AI prompt input | Cancel AI prompt |

## Standard Browser Shortcuts

These standard shortcuts also work in the editor:

| Shortcut | Action |
|----------|--------|
| `Ctrl+A` | Select all content |
| `Ctrl+C` | Copy selection |
| `Ctrl+V` | Paste from clipboard |
| `Ctrl+X` | Cut selection |
| `Ctrl+Z` | Undo (browser native) |
