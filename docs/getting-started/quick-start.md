# Quick Start

Create your first ebook in 5 minutes. This walkthrough covers the essential steps: setting metadata, writing content, and exporting.

## Step 1: Set Your Book's Metadata

1. Click the **Settings** icon (gear) in the sidebar, or press `Ctrl+M`
2. Fill in:
   - **Title** — your book's name
   - **Author** — the creator's name
   - **Publisher** — publishing entity (optional)
   - **Language** — e.g., `en`, `th`
   - **Description** — a brief synopsis
3. Optionally upload a **cover image** by clicking the upload area
4. Click **Save Changes**

## Step 2: Write Your First Chapter

The editor opens with a default chapter. Start editing the HTML content:

```html
<h1>My First Chapter</h1>
<p>This is the beginning of my story...</p>
<p>EpubCraft makes it easy to write and format ebooks.</p>
```

Use the **formatting toolbar** above the editor for quick formatting:
- **B** / **I** / **U** — Bold, italic, underline
- **H1**, **H2**, **H3** — Headings
- Alignment buttons — Left, center, right, justify
- List buttons — Bulleted and numbered lists

## Step 3: Add More Chapters

In the sidebar:
- Click **Add Chapter to End** to append a new chapter
- Hover over an existing chapter and click the **+** icon to insert one below it
- **Drag** the grip handle to reorder chapters

## Step 4: Style Your Book

Switch to CSS mode by clicking **Global CSS** in the sidebar. EpubCraft comes with three built-in templates:

1. **Noto Sans Thai** — Default, good for Thai and English text
2. **Serif** — Classic book look with Times New Roman
3. **Sarabun** — Thai-optimized with A4 formatting

Select a template from the **CSS Templates** dropdown in the toolbar, or write your own styles.

## Step 5: Preview

Click the **Preview** button in the toolbar (or use split view) to see your book rendered in a device mockup. The preview updates in real-time as you edit.

Click any element in the preview to jump to its corresponding line in the editor.

## Step 6: Export

1. Press `Ctrl+S` or click the **Export** button
2. Choose your format:
   - **Standard EPUB** — Ready for e-readers and distribution
   - **Project ZIP** — Backup format that can be re-imported
3. For Thai/Asian text, check **Optimize Line Breaking** to add zero-width spaces
4. Click **Export Now**

Your file downloads automatically. That's it — you've created an ebook with EpubCraft!
